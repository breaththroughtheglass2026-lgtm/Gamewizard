# 🎨 CanvasArtist: Mobile AI Game Creator
> **Dream. Code. Launch. (All from your phone).**

CanvasArtist is a specialized AI that builds mobile games for $0.

### 🚀 To-Do (Mobile Setup)
1. **API Key:** Go to [Groq](https://console.groq.com/) and get a free key.
2. **Secrets:** In this repo, go to **Settings > Secrets and variables > Actions**.
3. **Add Secret:** Name it `REACT_APP_AI_KEY` and paste your key.
4. **Build:** Run your deploy script to see it live at `yourname.github.io/repo`.
